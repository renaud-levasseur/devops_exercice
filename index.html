A propos

Ce projet est le back de l'applicatif Cocktail. Il s'agit d'un exercice de réalisation de projet.
L'API mise en place ici, est pour les playlists front de la chaine afin de suivre les vidéos

ATTENTION : La base de données est réinitialisée tous les jours à 6H du matin
Si vous avez fait des tests ne soyez pas surpris de ne pas les retrouver le lendemain ;)
